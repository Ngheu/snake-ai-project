# Snake AI Project - Đồ án Trí tuệ nhân tạo

Ứng dụng trí tuệ nhân tạo vào trò chơi Snake với 3 phương pháp:
- BFS (Breadth-First Search)
- Greedy Best-First Search
- Deep Q-Network với CNN (Convolutional Neural Network)

## Tính năng
- Hai bản đồ: Classic (không tường) và Maze (có tường ngẫu nhiên)
- Menu chọn thuật toán
- Huấn luyện và lưu model CNN-DQN
- Hiển thị đồ thị training realtime

## Yêu cầu
- Python 3.8+
- Pygame, Torch, Numpy, Matplotlib
